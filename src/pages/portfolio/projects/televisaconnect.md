**Connect** es una aplicación móvil para el área de ventas de televisa, el cual le permitiría al personal de ventas mostrar sus catálogos de horarios, además de información relevante para sus ventas.

Esta aplicación cuenta con las funcionalidades de notificaciones, buscador, apertura de contenido y reproducción de videos.

## 💡 Tecnologias utilizadas

- React
- React Native
- CSS
- JavaScript
