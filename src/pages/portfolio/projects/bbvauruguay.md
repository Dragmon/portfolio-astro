Participé en la construcción e implementación del nuevo flujo de contratación de cuentas para la aplicación de BBVA Uruguay, desarrollando una experiencia guiada que permite al cliente completar el proceso de contratación de forma clara e intuitiva.

El desarrollo se realizó a partir de los diseños definidos en Figma, utilizando las tecnologías y estándares internos del banco, siguiendo buenas prácticas de desarrollo, arquitectura y experiencia de usuario.

## 💡 Tecnologias utilizadas

- Lit Element
- Cells
- JavaScript
- SASS
- Bitbucket
- Git
- Web Componentes
