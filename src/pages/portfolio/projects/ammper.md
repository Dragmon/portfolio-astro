Ammper es una empresa que suministra energía eléctrica al país, además de ofrecer otros servicios relacionados con energía eléctrica.

Atenea es la plataforma de administración de facturas de Ammper, participe en la construcción de un nuevo flujo para la visualización y administración de las facturas que genera, utilizando las nuevas características del CFDI 4.0

## 💡 Tecnologias utilizadas

- React
- Next Js
- JavaScript
- Material UI
- Github
- GitLab
